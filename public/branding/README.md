# 브랜드 로고 이미지

이 폴더에는 LookMate의 브랜드 로고 파일들이 위치합니다.

## 필요한 파일

1. **lookmate-logo-soft.png** (파스텔톤 일러스트 버전)
   - 크기: 권장 500x150px (가로형) 또는 200x200px (정사각형)
   - 텍스트: "LOOKMATE – Your Style Best Friend"
   - 스타일: 파스텔 컬러, 부드러운 느낌

2. **lookmate-logo-neon.png** (네온 그라디언트 버전)
   - 크기: 권장 500x150px (가로형) 또는 200x200px (정사각형)
   - 텍스트: "LOOKMATE – Your Style Mate"
   - 스타일: 네온 그라디언트, 역동적인 느낌

3. **og-lookmate-soft.png** (OG 이미지, soft 버전)
   - 크기: 1200x630px (Open Graph 표준)
   - 소셜 미디어 공유용

4. **og-lookmate-neon.png** (OG 이미지, neon 버전)
   - 크기: 1200x630px (Open Graph 표준)
   - 소셜 미디어 공유용

## 임시 placeholder

실제 로고 이미지가 준비될 때까지 브라우저에서 이미지 깨짐 아이콘이 표시될 수 있습니다.
`config/branding.ts`에서 BRAND_VARIANT를 변경하면 자동으로 해당 버전의 로고를 사용합니다.
